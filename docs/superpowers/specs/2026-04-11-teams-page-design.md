# 球队页面设计

## 概述
创建球队介绍子页面，包含多个内容标签页，风格沿用 Spotify 暗黑设计系统。

## 页面结构
- 左侧固定导航（垂直）
- 右侧内容区（标签页切换）
- URL 路由：`/teams`

## 标签页内容
1. **球队介绍** - 简介、愿景、前景
2. **球员阵容** - 球员卡片网格
3. **荣誉墙** - 奖杯展示
4. **赛程** - 比赛列表

## 视觉风格
- 背景：#121212（深黑）
- 卡片：#181818
- 文字：#ffffff / #b3b3b3
- 按钮：Pill 形状
- 圆角：6-8px

## 交互
- 标签切换滑动动画
- 卡片悬停效果
- 平滑过渡

## 技术实现
- Vue Router 处理路由
- 组件化设计
- CSS 动画

## 修改文件
- `package.json` - 添加 vue-router
- `src/main.js` - 配置路由
- `src/App.vue` - 添加路由视图
- `src/views/TeamsPage.vue` - 球队页面主组件
- `src/components/` - 分割子组件
