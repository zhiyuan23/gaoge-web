# 球队页面重新设计

## 概述
重新设计球队介绍页面，使用标签页切换展示两个主队。

## 球队结构
- **高歌FC**（足球）
  - 皇家高歌（主题色 #F6E500）
  - 高歌国际（主题色 #1c69d4）
  - 高歌联（主题色 #DA291C）
- **热烈暖阳篮球俱乐部**（篮球）

## 页面结构
1. 顶部：返回按钮 + 品牌 + 导航
2. 标签栏：两个主队切换
3. 内容区：
   - 球队合影（大图横幅）
   - 球队简介
   - 子球队卡片（仅高歌FC）
   - 荣誉展示

## 技术实现
- Vue 3 + Tailwind CSS
- 标签页切换
- 响应式布局
